- Utilizamos o [pgAdmin4](https://www.pgadmin.org/download/pgadmin-4-windows/)

## Tecnologias utilizadas

* Java 21
* Spring Boot
* Spring Data JPA
* MySQL
* Docker
* Spring Cloud Open Feign
* ControllerAdvice & Problem Details
* Hibernate Validator
